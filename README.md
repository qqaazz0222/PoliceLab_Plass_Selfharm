# Dongguk Police Lab Selfharm Module

## Installation

```
git clone https://github.com/qqaazz0222/PoliceLab_Plass_Selfharm.git
cd PoliceLab_Plass_Selfharm
conda env create -f pyskl.yaml
conda activate pyskl
pip install -e .
```
